# p0-terraform-install

Deploy P0 integrations with Terraform

Supported integrations:

- Okta for AWS Account Federation
  - Login application: a native OIDC app for logging in from p0 cli
  - API integration application: a service app for managing group and application membership
  - An AWS Account federation app is also provided, however, this is a pre-existing app in customer's environment which may need changes.
